# App Preview Assets

This directory should contain the following app screenshot images:

## Required Files

1. **app-home.png** - Search screen with "Let's find your favorite car here"
   - Recommended size: 280x600px (iPhone mockup dimensions)
   - Format: PNG with transparency if needed

2. **app-detail.png** - Aston Martin Vantage detail screen
   - Recommended size: 280x600px (iPhone mockup dimensions)
   - Format: PNG with transparency if needed

3. **app-splash.png** - Drive Exotiq splash screen with logo
   - Recommended size: 280x600px (iPhone mockup dimensions)
   - Format: PNG with transparency if needed

## App Store Badges

The following badge images should be placed in `/public/images/`:

1. **app-store-badge.svg** - Apple App Store badge
   - Download from: https://developer.apple.com/app-store/marketing/guidelines/
   - Format: SVG preferred, or PNG at 120x40px

2. **google-play-badge.svg** - Google Play Store badge
   - Download from: https://play.google.com/intl/en_us/badges/
   - Format: SVG preferred, or PNG at 135x40px

## Notes

- All images will be optimized by Next.js Image component
- Images are lazy-loaded for performance
- Quality set to 90 for optimal balance between file size and visual quality
