# Event Images

This directory contains images for Drive Exotiq events.

## Required Files

### Denver Event
- **denver-event.jpg** (or .png) - Main image for the Denver Launch event
  - Recommended size: 1200x800px or larger
  - Format: JPG or PNG
  - Description: Audi S8 in desert landscape with person standing by car

## Usage

Images in this folder are referenced in:
- `/app/events/page.tsx` - Events page
- `/components/sections/Events.tsx` - Homepage events section

## Image Optimization

All images will be automatically optimized by Next.js Image component.
