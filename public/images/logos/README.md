# Drive Exotiq Logos

Place your logo files here:

## Required Files:
- `drive-exotiq-horizontal.svg` or `.png` - For header (optimized for dark background)
- `drive-exotiq-square.svg` or `.png` - For footer and square placements  
- `favicon.ico` - Browser tab icon

## Specifications:
- **Horizontal Logo:** ~200px width, transparent background
- **Square Logo:** 400x400px recommended
- **Favicon:** 32x32px or 16x16px
- **Format:** SVG preferred for scalability, PNG acceptable

## Current Status:
Header and Footer components are ready to display logos once files are added.
